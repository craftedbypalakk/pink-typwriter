# 🌸 Pink Typewriter

An interactive 3D pink Royal vintage typewriter built with Three.js + WebGL.

## Run locally
```bash
python3 -m http.server 8080
```
Then open http://localhost:8080/typewriter_pink.html

## Controls
- **Left-drag** — orbit
- **Scroll** — zoom  
- **Right-drag** — pan
